Project 40: Mixed Effects Models — Predicting Student Scores Across Schools

Fixed Effects Random Effects Intraclass Correlation statsmodels MixedLM

Difficulty: Hard  |  Builds on: Projects 9, 39

1 · Hypothetical Problem

  • An education board wants to predict student test scores based on study hours, parental education, and socioeconomic status
  • Data comes from 50 schools — students within the same school share unmeasured factors (teacher quality, resources, culture)
  • Standard linear regression (Project 9) treats all 5,000 students as independent — ignoring the school grouping
  • This violates the independence assumption and produces overconfident standard errors
  • Goal → use a mixed effects model that accounts for school-level variation while estimating student-level effects

2 · How to Approach It

  1. From Project 9: multiple regression assumes independent observations — violated when data is grouped
  2. From Project 39: GLMs extend the distribution family — mixed models extend the correlation structure
  3. Fixed effects: population-level relationships (study hours → score) — same for all schools
  4. Random effects: school-level deviations — each school has its own baseline (random intercept)
  5. Optionally: random slopes — the effect of study hours may differ by school

3 · How Can We Solve It

  1. Generate hierarchical student data nested within schools
  2. Show why ignoring grouping gives wrong standard errors
  3. Compute the Intraclass Correlation Coefficient (ICC) — how much variance is between schools?
  4. Fit a random intercept model using statsmodels MixedLM
  5. Fit a random intercept + random slope model
  6. Compare with pooled OLS and school-specific OLS (no pooling)

4 · What Should Do

  • ✅ Compute ICC first — if it's near zero, grouping doesn't matter; if high, mixed models are essential
  • ✅ Start with random intercepts only — add random slopes only if justified
  • ✅ Use REML (Restricted Maximum Likelihood) for unbiased variance estimates
  • ✅ Check residuals at both levels — student-level and school-level
  • ✅ Compare AIC/BIC across models to select the best structure

5 · What Should NOT Do

  • ❌ Do NOT ignore the grouping structure — standard errors will be too small, p-values too optimistic
  • ❌ Do NOT add school as a fixed effect with 50 dummy variables — wastes degrees of freedom and can't generalize
  • ❌ Do NOT fit separate models per school — small schools have too few observations for reliable estimates
  • ❌ Do NOT add random slopes without checking if they improve the model — overfitting risk
  • ❌ Do NOT confuse fixed and random effects — fixed effects are estimated; random effects are predicted (BLUPs)

6 · When to Use What

  • 🔹 Mixed effects model → when data has a natural grouping (students in schools, patients in hospitals)
  • 🔹 Random intercept → when groups differ in baseline level but not in slopes
  • 🔹 Random intercept + slope → when the effect of a predictor varies across groups
  • 🔹 Pooled OLS (Project 9) → when there's no grouping or ICC ≈ 0
  • 🔹 Fixed effects (dummy variables) → when you have few groups and want to control for group-level confounders

7 · When NOT to Use

  • 🚫 When there's no natural grouping in the data — use standard regression
  • 🚫 When you have very few groups (< 5) — random effects estimates are unreliable
  • 🚫 When ICC is near zero — grouping doesn't matter, OLS is fine
  • 🚫 When the research question is about specific groups — use fixed effects instead

8 · How It Only Works — Internal Mechanics

The Mixed Model Equation

  • y_ij = β₀ + β₁x_ij + u_j + ε_ij
  • β₀, β₁ = fixed effects (population-level intercept and slope)
  • u_j ~ N(0, σ²_u) = random intercept for school j
  • ε_ij ~ N(0, σ²_e) = residual error for student i in school j
  • Each school's intercept = β₀ + u_j — shifted up or down from the population mean

Intraclass Correlation (ICC)

  • ICC = σ²_u / (σ²_u + σ²_e)
  • Proportion of total variance that's between groups
  • ICC = 0 → no group effect; ICC = 1 → all variation is between groups
  • ICC > 0.05 typically justifies using a mixed model

Shrinkage / Partial Pooling

  • Mixed models "shrink" group estimates toward the population mean
  • Small groups (few students) → more shrinkage (borrow strength from other schools)
  • Large groups → less shrinkage (enough data to estimate reliably)
  • This is the key advantage over no-pooling (separate models) and complete pooling (ignoring groups)

REML vs ML Estimation

  • ML: maximizes the full likelihood — biased variance estimates (like dividing by n instead of n-1)
  • REML: integrates out fixed effects first — unbiased variance estimates
  • Use REML for final model; use ML when comparing models with different fixed effects

9 · Common Pitfalls and Failure Modes

  • ⚠️ Ignoring grouping — inflated Type I error rates; you'll find "significant" effects that aren't real
  • ⚠️ Too many random effects — convergence issues; start simple and add complexity
  • ⚠️ Few groups — with < 10 groups, random effects variance is poorly estimated
  • ⚠️ Singular fit — random effects variance estimated at zero; the model is overparameterized
  • ⚠️ Confusing levels — student-level predictors vs school-level predictors need different treatment

10 · Scalability Considerations

  • statsmodels MixedLM handles thousands of groups and millions of observations
  • Computation scales as O(n × p²) per iteration — similar to standard regression
  • Random slopes increase complexity — each random slope adds a variance parameter to estimate
  • For very complex random effects structures, consider lme4 in R via rpy2 or Bayesian alternatives

11 · Interpretability Considerations

  • ✅ Fixed effects are interpreted like standard regression coefficients
  • ✅ ICC is intuitive: "25% of score variation is between schools"
  • ✅ Random effects (BLUPs) show which schools are above/below average
  • ⚠️ Random effects are predictions, not estimates — they have uncertainty too
  • ⚠️ The shrinkage concept can confuse stakeholders — "why is my school's estimate pulled toward the mean?"

12 · Complete Working Code

import numpy as np
import pandas as pd
import statsmodels.api as sm
import statsmodels.formula.api as smf
from sklearn.linear_model import LinearRegression
import matplotlib
matplotlib.use('Agg')
import matplotlib.pyplot as plt

np.random.seed(42)

# ─── Step 1: Generate hierarchical student data ───
n_schools = 50
students_per_school = np.random.randint(60, 150, n_schools)
n_total = students_per_school.sum()

school_ids = np.repeat(np.arange(n_schools), students_per_school)
school_intercepts = np.random.normal(0, 8, n_schools)  # school-level variation
school_slopes = np.random.normal(0, 0.5, n_schools)    # slope variation

study_hours = np.random.uniform(1, 10, n_total)
parent_edu = np.random.choice([0, 1, 2], n_total, p=[0.3, 0.45, 0.25])
ses_score = np.random.normal(50, 10, n_total)

# True model: score = 50 + 3*hours + 5*parent_edu + 0.2*ses + school_effect + noise
true_intercept = 50
true_hours_effect = 3.0
scores = (true_intercept
          + (true_hours_effect + school_slopes[school_ids]) * study_hours
          + 5 * parent_edu + 0.2 * ses_score
          + school_intercepts[school_ids]
          + np.random.normal(0, 10, n_total))

df = pd.DataFrame({
    'school': school_ids, 'study_hours': study_hours,
    'parent_edu': parent_edu, 'ses_score': ses_score, 'score': scores
})

print("=== Hierarchical Student Data ===")
print(f"  Schools: {n_schools}, Total students: {n_total}")
print(f"  Students per school: {students_per_school.min()}–{students_per_school.max()}")
print(f"  Score range: {scores.min():.0f}–{scores.max():.0f}")

# ─── Step 2: Compute ICC (null model) ───
null_model = smf.mixedlm("score ~ 1", df, groups=df["school"]).fit(reml=True)
var_school = null_model.cov_re.iloc[0, 0]
var_resid = null_model.scale
icc = var_school / (var_school + var_resid)

print(f"\n=== Intraclass Correlation ===")
print(f"  Between-school variance: {var_school:.2f}")
print(f"  Within-school variance:  {var_resid:.2f}")
print(f"  ICC: {icc:.3f} ({icc:.1%} of variance is between schools)")
print(f"  → {'Mixed model justified' if icc > 0.05 else 'OLS may suffice'}")

# ─── Step 3: Pooled OLS (ignoring schools) ───
ols_model = smf.ols("score ~ study_hours + parent_edu + ses_score", df).fit()
print(f"\n=== Pooled OLS (Ignoring Schools) ===")
print(f"  study_hours: β={ols_model.params['study_hours']:.3f}, "
      f"SE={ols_model.bse['study_hours']:.3f}, p={ols_model.pvalues['study_hours']:.4f}")
print(f"  R²: {ols_model.rsquared:.4f}")

# ─── Step 4: Random Intercept Model ───
ri_model = smf.mixedlm(
    "score ~ study_hours + parent_edu + ses_score",
    df, groups=df["school"]
).fit(reml=True)

print(f"\n=== Random Intercept Model ===")
print(f"  Fixed Effects:")
for param in ['Intercept', 'study_hours', 'parent_edu', 'ses_score']:
    print(f"    {param:<15}: β={ri_model.fe_params[param]:.3f}, "
          f"SE={ri_model.bse_fe[param]:.3f}")
print(f"  Random Effects:")
print(f"    School intercept SD: {np.sqrt(ri_model.cov_re.iloc[0,0]):.3f}")
print(f"    Residual SD: {np.sqrt(ri_model.scale):.3f}")

# ─── Step 5: Random Intercept + Random Slope Model ───
ris_model = smf.mixedlm(
    "score ~ study_hours + parent_edu + ses_score",
    df, groups=df["school"],
    re_formula="~study_hours"
).fit(reml=True)

print(f"\n=== Random Intercept + Slope Model ===")
print(f"  Fixed Effects:")
for param in ['Intercept', 'study_hours', 'parent_edu', 'ses_score']:
    print(f"    {param:<15}: β={ris_model.fe_params[param]:.3f}, "
          f"SE={ris_model.bse_fe[param]:.3f}")
print(f"  Random Effects Covariance:")
print(f"    {ris_model.cov_re}")

# ─── Step 6: Compare models ───
print(f"\n=== Model Comparison ===")
print(f"  {'Model':<30} {'Log-Lik':>10} {'AIC':>10} {'BIC':>10}")
print(f"  {'-'*62}")
print(f"  {'Pooled OLS':<30} {ols_model.llf:>10.1f} {ols_model.aic:>10.1f} {ols_model.bic:>10.1f}")

# ─── Step 7: Extract and visualize random effects ───
re_values = ri_model.random_effects
school_re = [re_values[i]['Group'] for i in range(n_schools)]

print(f"\n=== School Random Effects (Top 5 / Bottom 5) ===")
re_df = pd.DataFrame({'school': range(n_schools), 'random_intercept': school_re})
re_df = re_df.sort_values('random_intercept')
print("  Bottom 5 (below average):")
for _, row in re_df.head().iterrows():
    print(f"    School {int(row.school)}: {row.random_intercept:+.2f}")
print("  Top 5 (above average):")
for _, row in re_df.tail().iterrows():
    print(f"    School {int(row.school)}: {row.random_intercept:+.2f}")

# ─── Step 8: Visualization ───
fig, axes = plt.subplots(2, 2, figsize=(14, 12))

# School random intercepts
ax = axes[0, 0]
re_sorted = re_df.sort_values('random_intercept')
colors = ['#f85149' if v < 0 else '#3fb950' for v in re_sorted.random_intercept]
ax.barh(range(n_schools), re_sorted.random_intercept, color=colors, alpha=0.7)
ax.set_xlabel('Random Intercept (deviation from mean)')
ax.set_ylabel('School (sorted)')
ax.set_title('School-Level Random Effects')
ax.axvline(0, color='white', linewidth=0.5)
ax.grid(True, alpha=0.2, axis='x')

# Shrinkage plot
ax = axes[0, 1]
school_means = df.groupby('school')['score'].mean().values
school_counts = df.groupby('school').size().values
mixed_means = ri_model.fe_params['Intercept'] + np.array(school_re)
grand_mean = df['score'].mean()
ax.scatter(school_means, mixed_means, c=school_counts, cmap='viridis', s=30, alpha=0.8)
ax.plot([school_means.min(), school_means.max()],
        [school_means.min(), school_means.max()], '--', color='#f85149', alpha=0.5, label='No shrinkage')
ax.axhline(grand_mean, color='#58a6ff', linestyle=':', alpha=0.5, label='Grand mean')
ax.set_xlabel('Raw School Mean')
ax.set_ylabel('Mixed Model Estimate')
ax.set_title('Shrinkage: Raw vs Mixed Model Estimates')
ax.legend(fontsize=8)
ax.grid(True, alpha=0.2)
plt.colorbar(ax.collections[0], ax=ax, label='N students')

# School-specific regression lines (sample of 10)
ax = axes[1, 0]
sample_schools = np.random.choice(n_schools, 10, replace=False)
hours_range = np.linspace(1, 10, 50)
for s in sample_schools:
    mask = df.school == s
    ax.scatter(df.loc[mask, 'study_hours'], df.loc[mask, 'score'], alpha=0.15, s=5)
    intercept = ri_model.fe_params['Intercept'] + re_values[s]['Group']
    slope = ri_model.fe_params['study_hours']
    ax.plot(hours_range, intercept + slope * hours_range, linewidth=1.5, alpha=0.7)
ax.set_xlabel('Study Hours')
ax.set_ylabel('Score')
ax.set_title('School-Specific Regression Lines (10 schools)')
ax.grid(True, alpha=0.2)

# OLS vs Mixed model SE comparison
ax = axes[1, 1]
params = ['study_hours', 'parent_edu', 'ses_score']
ols_se = [ols_model.bse[p] for p in params]
mixed_se = [ri_model.bse_fe[p] for p in params]
x = np.arange(len(params))
ax.bar(x - 0.15, ols_se, 0.3, color='#f85149', alpha=0.7, label='OLS SE')
ax.bar(x + 0.15, mixed_se, 0.3, color='#3fb950', alpha=0.7, label='Mixed SE')
ax.set_xticks(x)
ax.set_xticklabels(params)
ax.set_ylabel('Standard Error')
ax.set_title('Standard Errors: OLS vs Mixed Model')
ax.legend()
ax.grid(True, alpha=0.2, axis='y')

plt.tight_layout()
plt.savefig('project_40_mixed_effects.png', dpi=100, bbox_inches='tight')
print(f"\n✅ Mixed effects analysis saved as project_40_mixed_effects.png")

13 · Key Takeaway + Bridge to Next Project

Key Takeaway

  • Mixed effects models handle grouped/hierarchical data by separating fixed (population) and random (group) effects
  • ICC tells you how much variance is between groups — high ICC demands a mixed model
  • Partial pooling (shrinkage) borrows strength across groups — better than ignoring groups or fitting separate models
  • Standard errors from OLS are wrong when data is grouped — mixed models correct this
